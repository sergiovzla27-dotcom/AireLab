# AireLab 🌬️

**Sistema de monitoreo de calidad del aire y alertas para Navojoa, Sonora**

AireLab es un proyecto que analiza la relación entre los niveles de PM2.5 y los padecimientos respiratorios en Navojoa, Sonora, con el fin de generar un sistema de alerta para la población vulnerable.

---

## 📋 Descripción del proyecto

Este proyecto busca identificar la correlación entre la contaminación del aire (PM2.5) y el aumento de enfermedades respiratorias en Navojoa, mediante:

- **Recolección automática de datos de PM2.5** desde la API de OpenWeather
- **Encuesta ciudadana** de percepción de síntomas respiratorios
- **Análisis estadístico de correlación** entre ambas variables
- **Sistema de alertas** vía WhatsApp para población vulnerable (niños menores de 5 años, adultos mayores de 60 años y personas con enfermedades respiratorias previas)
